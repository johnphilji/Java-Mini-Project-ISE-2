# Microfinance Loan Tracker

## Project Summary
The Microfinance Loan Tracker is a software application designed to manage and track microfinance loans efficiently. It provides tools for loan management, borrower tracking, repayment schedules, and reporting, making it easier for microfinance institutions to streamline their operations.

## Tech Stack
- **Programming Language**: Java
- **Database**: MySQL
- **Frontend**: JavaFX
- **Build Tool**: Maven
- **Version Control**: Git

## Features
- **Loan Management**: Create, update, and delete loan records.
- **Borrower Profiles**: Maintain detailed borrower information.
- **Repayment Tracking**: Monitor repayment schedules and overdue payments.
- **Reporting**: Generate reports on loan statuses and borrower performance.
- **Search and Filter**: Easily search and filter loans and borrowers.

## How to Run
1. Clone the repository.
2. Set up the database using the provided schema.
3. Configure the application properties.
4. Build and run the project using Maven.
